# reindexBags
A script to reindex all active rosbag files in specified folder
<br>
<br>
![Alt Text](reindexBags_rate5.gif)

# Usage
make the python script executable:
<pre><code>
chmod +x reindexBags.py
</pre></code>
execute the script:
<pre><code>
./reindexBags -s /path_to_folder/
</pre></code>
