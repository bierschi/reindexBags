# reindexBags
A script to reindex all rosbag files in specified folder

# Usage
### python
make the python script executable:
<pre><code>
chmod +x reindexBags.py
</pre></code>
execute the script:
<pre><code>
./reindexBags -s /path_to_folder/
</pre></code>
